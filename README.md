# Fanta Drink

An **interactive Fanta landing page** built with vanilla HTML, CSS and JavaScript — featuring a scroll-animated can, floating canvas bubbles and a real-time flavor switcher, zero framework.

## Features

**Scroll-Animated Can** – The Fanta can moves, rotates and scales across sections via GSAP ScrollTrigger keyframes  
**Drop-In Entry Animation** – Can falls from the top on page load with a physical bounce effect  
**Floating Canvas Bubbles** – 45 dynamically generated bubbles with mouse repulsion, radial gradient and light reflection  
**Flavor Switcher** – 3 themes (Orange, Grape, Lemon) that instantly update background, shadows, CSS variables and bubble colors  
**Smooth Theme Transitions** – All flavor transitions interpolated through GSAP for a fluid visual experience  
**Glassmorphism Panels** – Backdrop-blur surfaces with luminescent hover borders  
**Mobile Hamburger Menu** – Icon animation, scroll-close behavior and blurred overlay  
**Full Responsive** – 3 breakpoints: Desktop (≥1024px), Tablet (768–1023px), Mobile (<768px)  

## Technologies Used

- **HTML5** – Semantic structure and inline custom SVG logo
- **CSS3** – Keyframe animations, CSS variables, glassmorphism and media queries
- **Vanilla JavaScript** – Animation logic, Canvas bubbles and flavor switching
- **GSAP 3 + ScrollTrigger** – Scroll-driven can animation timeline
- **Tailwind CSS** – Utility layout via CDN
- **Canvas API** – Floating bubble system with mouse interaction
